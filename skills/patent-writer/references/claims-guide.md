## 基本概念

### 权利要求书的作用
权利要求书是专利的"心脏"，直接决定专利保护范围的大小和质量。

### 两种类型对比

| 特性 | 独立权利要求 | 从属权利要求 |
|------|-------------|-------------|
| 独立性 | 可独立存在 | 必须引用其他权利要求 |
| 范围 | 最宽的保护范围 | 较窄的保护范围 |
| 内容 | 全部必要技术特征 | 必要特征 + 附加特征 |
| 作用 | 定义核心发明 | 提供保护层次和备选方案 |

## 独立权利要求撰写

### 核心原则
1. **包含全部必要技术特征**：解决技术问题不可缺少的特征
2. **避免非必要特征**：将可选特征放入从属权利要求
3. **使用上位概念**：扩展专利保护范围

### 撰写前准备
1. 从机器/控制器角度梳理技术方案的所有执行动作
2. 用一句话概括与现有技术的核心区别技术特征
3. 确定必要技术特征清单

### 标准结构
```
一种[主题名称]，其特征在于，包括：
[技术特征1]；
[技术特征2]；
...
[技术特征N]。
```

### 撰写模板
```
权利要求1：
一种[产品/方法名称]，其特征在于，包括：
[前序部分：现有技术特征]；
其特征在于：
[特征部分：区别于现有技术的技术特征]。
```

### 上位概括策略
| 具体表述 | 上位概括 |
|----------|----------|
| 螺丝钉 | 紧固件 |
| 电动机 | 驱动装置 |
| 钢板 | 金属板材 |
| 手机 | 移动终端 |
| 蓝牙 | 无线通信模块 |

## 从属权利要求撰写

### 引用规范
1. 首先引用其依赖的权利要求编号
2. 使用"根据权利要求X所述的…"格式
3. 只能引用在前的权利要求

### 撰写模板
```
权利要求N：
根据权利要求[M]所述的[主题名称]，其特征在于，[附加技术特征]。
```

### 技术特征安排
- 将**非必要技术特征**写入从属权利要求
- 按重要性层次排列
- 形成递进式保护

### 多重从属权利要求
```
权利要求N：
根据权利要求[M1]、[M2]或[M3]任一项所述的[主题名称]，其特征在于，[附加技术特征]。
```

**注意**：不能使用"和"进行累积引用

## 撰写技巧与策略

### 保护范围优化策略

```
                    独立权利要求1
                    (最宽保护范围)
                         │
          ┌──────────────┼──────────────┐
          ▼              ▼              ▼
     权利要求2      权利要求3      权利要求4
     (实施例A)      (实施例B)      (实施例C)
          │              │
          ▼              ▼
     权利要求5      权利要求6
     (优选例A1)     (优选例B1)
```

### 常见布局模式
1. **产品+方法**：独立权利要求分别保护产品和方法
2. **多层保护**：独立权利要求 + 多级从属权利要求
3. **备选方案**：多个并列的从属权利要求

### 创造性体现
- 在优选实施例中找出最具创造性的关键创新点
- 逐层限定，形成保护层次
- 预留规避设计的应对方案

## 常见问题与示例

### 问题1：独立权利要求过窄
**错误示例**：
```
一种用铝合金制造的自行车车架...
```
**正确示例**：
```
一种自行车车架，其特征在于，包括：
主管件；以及
与所述主管件连接的支管件；
其中，所述主管件和支管件通过焊接方式连接。
```

### 问题2：从属权利要求引用错误
**错误示例**：
```
权利要求3：根据权利要求4所述的...
```
（引用了后面的权利要求）

**正确示例**：
```
权利要求3：根据权利要求1或2所述的...
```

### 问题3：累积引用
**错误示例**：
```
根据权利要求1和2所述的...
```

**正确示例**：
```
根据权利要求1或2所述的...
```

### 完整示例
```
权利要求1：
一种智能温控系统，其特征在于，包括：
温度传感器，用于检测环境温度；
控制单元，与所述温度传感器连接；
执行单元，与所述控制单元连接，用于调节环境温度；
其中，所述控制单元根据所述温度传感器检测的温度与预设阈值的比较结果，控制所述执行单元的运行状态。

权利要求2：
根据权利要求1所述的智能温控系统，其特征在于，所述温度传感器为数字式温度传感器。

权利要求3：
根据权利要求1所述的智能温控系统，其特征在于，还包括：
通信模块，与所述控制单元连接，用于与外部设备进行数据通信。

权利要求4：
根据权利要求3所述的智能温控系统，其特征在于，所述通信模块包括WiFi模块、蓝牙模块或Zigbee模块中的至少一种。
```
