## 基础篇（系统是什么）

### Ch.1 信息系统与信息技术发展（约 3 题）
- 信息化/信息系统内涵与特征；信息系统发展（**诺兰模型六阶段**：初始→蔓延→控制→集成→数据管理→成熟）
- 计算机软硬件、网络、数据存储与数据库、信息安全基础
- **新一代信息技术（高频）**：物联网（感知/网络/应用层）、区块链（去中心化/智能合约）、云计算、大数据（**5V**：Volume/Velocity/Variety/Veracity/Value）、人工智能、边缘计算、数字孪生
- 陷阱：各技术"特征"易被错位出题，须记准每个技术的定义性特征

### Ch.2 数字中国与数智化发展（约 3 题）
- 数字转型（驱动因素/基本原理/**GB/T 数字化转型参考架构**）
- 数字中国 = 数字经济 + 数字政府 + 数字社会 + 数字生态
- 数字经济、数字政府（一网通办/一网统管/一网协同）、数字社会（智慧城市/数字乡村/数字生活）、数据要素市场、网络安全保护

### Ch.3 系统科学与哲学方法论（约 3 题，★新版重点）
- **老三论**：系统论、信息论、控制论
- **新三论**：耗散结构理论、协同论、突变论
- 矛盾论、实践论、复杂系统论
- 每个理论记"主要思想 + 在系统规划与管理中的应用"两层；考应用对应关系

## 方法篇（怎么规划）

### Ch.4 信息系统规划（约 3 题）
- ISP 主要内容：战略、框架、组织、技术、任务、资源、保障七大体系
- **常用方法（高频易混）**：SST 战略目标集转移法、BSP 企业系统规划法、CSF 关键成功因素法、VCA 价值链分析法、Zachman 框架
- 工作要点：需求挖掘、场景化模型、深度诊断、整体与专项规划、持续改进

### Ch.5 应用系统规划（约 3 题）
- 生命周期选择、体系结构/接口/数据/构件定义
- 主要过程：初步调研→可行性研究→详细调研→系统分析→系统设计
- 方法：**TOGAF**（企业架构）、**SOA** 面向服务架构、应用系统组合法、软件工厂

### Ch.6 云资源规划（约 4 题）
- **云服务模式**：IaaS / PaaS / SaaS
- **云部署模式**：公有云 / 私有云 / 混合云 / 社区云
- 云计算特征（内部/外部）、计算资源规划、存储资源规划、云数据中心

### Ch.7 网络环境规划（约 3 题）
- **OSI 七层**（物/数/网/传/会/表/应）vs TCP/IP 四层
- IPv4 vs IPv6、广域网/局域网/无线网技术
- 4G/5G、NB-IoT、WiFi、网络拓扑、机房建设、综合布线、网络安全设计

### Ch.8 数据资源规划（约 3 题）
- 数据资源规划方法（稳定信息过程/稳定信息结构/指标能力）
- 数据架构（数据模型/数据流设计/传统 vs 现代数据架构）
- 数据标准化（**元数据**/数据元/分类编码标准化）
- 数据管理（**数据治理**/数据质量/数据安全）

### Ch.9 信息安全规划（约 3 题）
- 安全威胁、安全规划原则与注意事项
- 安全架构（商业应用安全架构 SABSA、信息系统安全保障模型）
- 组织体系 / 管理体系 / 技术体系 / 运营体系规划

### Ch.10 云原生系统规划（约 3 题）
- 云原生核心技术：**容器（Docker）、微服务、DevOps、服务网格 Service Mesh、声明式 API**
- 设计原则、架构模式、建设规划

## 高频易混淆点辨析

| 易混项 | 关键区分 |
|--------|---------|
| SST / BSP / CSF / VCA | SST 战略目标自上而下转移；BSP 自下而上数据类；CSF 聚焦关键成功因素；VCA 按价值链活动分析 |
| IaaS / PaaS / SaaS | IaaS=基础设施(虚机)、PaaS=平台(开发运行环境)、SaaS=软件(成品应用) |
| 老三论 / 新三论 | 老三论=系统论+信息论+控制论；新三论=耗散结构+协同论+突变论 |
| 公有/私有/混合/社区云 | 公有=第三方公开；私有=单一组织；混合=组合；社区=若干共同需求组织共享 |
| 元数据 / 数据元 | 元数据=描述数据的数据；数据元=可识别的数据单元 |
| SLA / OLA / UC | SLA=客户与服务商外部协议；OLA=内部支撑协议；UC=与外部供应商合同 |
| ITIL / ITSS / COBIT | ITIL=国际IT服务管理最佳实践；ITSS=中国信息技术服务标准；COBIT=IT治理框架 |
| 过程组 / 绩效域 | PMBOK6 旧=5过程组+10知识领域；PMBOK7 新=8绩效域+12原则（价值驱动） |

## 记忆口诀汇总
- **PPTR 四要素**：人(People)过(Process)技(Technology)源(Resource)
- **老三论**：系信控（系统、信息、控制）
- **新三论**：耗协突（耗散、协同、突变）
- **OSI 七层**：物数网传会表应
- **PDCA 服务改进**：策划-实施-检查-处置
- **5V 大数据**：多快杂准值（Volume/Velocity/Variety/Veracity/Value）
- **诺兰六阶段**：初蔓控集数熟
