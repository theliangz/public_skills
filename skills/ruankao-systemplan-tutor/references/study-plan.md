# 备考策略与备考资料

制定复习计划、推荐资料用法、做备考咨询时加载。

## 一、四阶段备考策略

| 阶段 | 周期参考 | 核心任务 | 每日时长 |
|------|---------|---------|---------|
| **① 入门摸底** | 1 周 | 做摸底测试卷，建立整体认知，明确薄弱点 | 2h |
| **② 网课学习**（重点） | 12-14 周 | 精讲视频 + 章节练习，逐章过关，夯实基础 | 2-3h |
| **③ 串讲突破** | 2 周 | 考前冲刺串讲，建立知识点联系，专项突破案例+论文 | 3h |
| **④ 刷题模考** | 3-4 周 | 历年真题 + 模考 ≥5 套，培养题感，查漏补缺 | 3h |

**日常必做**：网课/录播 + 章节练习（其余可选）。坚持每日 2-3 小时是关键。

## 二、按剩余时间排期（以目标 2026 年下半年 11 月为例，约 20 周）

> 若剩余时间不同，按比例压缩/拉伸各阶段，但**阶段②（网课学习）与阶段④（刷题）不可压缩**。

```
第 1 周        摸底测试 + 通读思维导图/考情分析，建立框架
第 2-14 周     逐章精讲（基础篇→方法篇→能力篇→实践篇）+ 章节练习
               · 能力篇(Ch.11-17)多花时间，是案例论文核心
               · 第 12 周起准备论文项目背景
第 15-16 周    串讲 + 案例分析专项 + 论文专项（每周 1 篇论文练习）
第 17-20 周    真题(2024/2025) + 两期模考 + 默写本 + 考前几页纸
               · 模考要计时，模拟连考 4 小时
第 20 周末     考前 20 问 + 考点自查清单，临门一脚
```

## 三、各科复习重心

- **综合知识**：贯穿全程。阶段②靠章节练习巩固，阶段④靠真题+模考。**重点是把教材原文读熟**（应对挖空/错位出题）。
- **案例分析**：阶段②后期开始，重点练 **PPTR + 计算题**。阶段③④集中突破。
- **论文**：阶段②末期准备 1-2 个项目背景，阶段③④每周写 1 篇。**早准备、多练手**。

## 四、备考资料索引（按需 OCR）

若用户本机备有备考资料（多为扫描版 PDF），可读取辅助复习。**先向用户确认资料目录路径**，按需 OCR（Read 指定页 → 图像分析提取）。以下为常见备考资料分类，供定位参考（用户实际文件名可能不同）：

### 知识核心类（替代/精简教材，背诵主力）
| 资料类型 | 用途 |
|------|------|
| 知识点集锦 | 浓缩精华，全程必看 |
| 核心宝典 | 重点提炼，阶段②③用 |
| 三色笔记 | 三色标注重难点 |
| 知识点速记（50/100条） | 背诵速记 |
| 默写本 | 自测默写，阶段④用 |

### 框架导览类（建立体系、考前速览）
| 资料类型 | 用途 |
|------|------|
| 思维导图 | 建立知识体系，阶段①看 |
| 考情分析 | 命题规律与分值分布 |
| 学习计划 | 排期参考 |
| 考点自查清单 | 阶段④查漏 |
| 考前几页纸 / 考前N问 | 考前临门速览 |

### 案例与论文类
| 资料类型 | 用途 |
|------|------|
| 论文范文 | 论文模仿学习 |
| 易混淆知识点 | 辨析易错点 |

### 真题与模考类（阶段④主力）
| 资料类型 | 用途 |
|------|------|
| 历年真题（含答案） | 真题实战 |
| 经典100题 | 高频题练习 |
| 摸底测试卷 | 阶段①摸底 |
| 各期模考试卷（综合/案例/论文） | 计时模考 |

## 五、使用建议（给导师侧）
- 用户问某章知识 → 结合知识地图讲解；若用户备有知识点集锦/核心宝典等资料，读取对应章节
- 用户做真题 → 若用户提供真题资料，逐题讲解；否则按命题规律出模拟题
- 用户练论文 → 用论文模板套写；若用户备有论文范文，参考其结构
- 用户要计划 → 按其剩余时间套用上方排期模板
