# 股票分析 Skill

这是一个综合性的股票分析工具，帮助您筛选可投资股票、生成投资建议并提供风险预警。

## 核心功能

### 1. 股票筛选
- **短期筛选**：题材热点、板块利好、放量交易、技术指标（MACD、KDJ、RSI）、资金流向、市场情绪
- **长期筛选**：财务数据、行业前景、公司竞争力、估值水平（PE、PB、DCF）

### 2. 投资建议生成
- 基于多维度分析给出买入/持有/卖出建议
- 提供详细的投资理由和风险提示

### 3. 风险预警
- 识别潜在投资风险
- 提供预警信息和建议应对措施

## 工作流程

### 步骤1：获取最新资讯
使用以下工具获取市场资讯：
- WebReader：从财经网站获取最新资讯（同花顺、中财网、东方财富）
- WebSearch：搜索特定股票的最新信息

**主要资讯来源：**
- 同花顺：https://www.10jqka.com.cn/
- 中财网：https://www.cfi.cn/index.aspx?client=pc
- 东方财富行情中心：https://quote.eastmoney.com/center/
- 东方财富网：https://www.eastmoney.com/

### 步骤2：股票筛选分析
根据用户需求（短期/长期）使用相应的筛选标准：

**短期筛选流程：**
1. 识别当前热点题材和板块
2. 分析板块利好消息
3. 筛选放量交易的个股
4. 分析技术指标（MACD、KDJ、RSI等）
5. 分析资金流向
6. 评估市场情绪

**长期筛选流程：**
1. 分析公司财务数据（营收、利润、ROE等）
2. 评估行业发展前景
3. 分析公司竞争力和护城河
4. 进行估值分析（PE、PB、DCF等）

### 步骤3：深度分析
对于筛选出的股票，进行深度分析：
- 通过网络搜索获取更详细的公司信息
- 分析最新公告和新闻
- 评估投资价值和风险

### 步骤4：生成报告
根据分析结果生成：
- **详细分析报告**：包含评分、投资理由、风险分析、操作建议
- **简洁表格清单**：列出股票代码、名称、价格、核心买入理由、风险等级

## 使用示例

**示例1：短期热点筛选**
```
用户：帮我筛选一下今天的热点股票，关注短期机会
工作流程：
1. 使用WebReader获取同花顺、东方财富的热点资讯
2. 识别热点板块和题材
3. 筛选放量交易的个股
4. 分析技术指标和资金流向
5. 生成分析报告和表格清单
```

**示例2：长期价值投资分析**
```
用户：分析一下贵州茅台的长期投资价值
工作流程：
1. 使用WebSearch搜索贵州茅台的最新信息
2. 分析财务数据和行业地位
3. 进行估值分析
4. 评估长期投资价值
5. 生成详细分析报告
```

**示例3：风险预警**
```
用户：我持有XX股票，帮我分析一下风险
工作流程：
1. 获取该股票的最新资讯
2. 分析潜在风险因素
3. 评估风险等级
4. 提供应对建议
5. 生成风险预警报告
```

## 注意事项

1. **投资有风险**：所有分析和建议仅供参考，不构成投资建议
2. **数据时效性**：确保使用最新的市场数据进行分析
3. **多维度分析**：避免单一指标决策，综合考虑多个因素
4. **风险控制**：始终关注风险，设置止损点
5. **市场变化**：市场情况瞬息万变，需要及时跟踪调整
