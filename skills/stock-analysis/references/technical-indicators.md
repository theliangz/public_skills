# 技术指标分析参考

## MACD指标（指数平滑异同移动平均线）

### 计算方法
- 快速EMA（通常12日）
- 慢速EMA（通常26日）
- DIF = EMA(12) - EMA(26)
- DEA = DIF的EMA（通常9日）
- MACD = (DIF - DEA) × 2

### 买卖信号
**买入信号：**
- DIF上穿DEA（金叉）
- MACD柱状线由负转正
- DIF和DEA均在0轴上方形成金叉

**卖出信号：**
- DIF下穿DEA（死叉）
- MACD柱状线由正转负
- DIF和DEA均在0轴下方形成死叉

### 注意事项
- 0轴上方为多头市场，下方为空头市场
- 柱状线持续放大说明趋势增强
- 适合趋势行情，震荡市容易产生假信号

## KDJ指标（随机指标）

### 计算方法
- RSV = (当日收盘价 - N日内最低价) / (N日内最高价 - N日内最低价) × 100
- K = RSV的M1日移动平均
- D = K的M2日移动平均
- J = 3K - 2D

### 买卖信号
**买入信号：**
- K线上穿D线（金叉）
- K、D值低于20（超卖区）
- J线大于100且持续上升

**卖出信号：**
- K线下穿D线（死叉）
- K、D值高于80（超买区）
- J线小于0且持续下降

### 注意事项
- 超买超卖区间：80以上超买，20以下超卖
- 50为多空分界线
- J线反应最敏感，K线次之，D线最稳定
- 适合震荡行情

## RSI指标（相对强弱指标）

### 计算方法
- RSI = 100 - 100 / (1 + RS)
- RS = N日内收盘涨幅均值 / N日内收盘跌幅均值

### 买卖信号
**买入信号：**
- RSI低于30（超卖）
- RSI形成底背离
- RSI由下向上突破50

**卖出信号：**
- RSI高于70（超买）
- RSI形成顶背离
- RSI由上向下突破50

### 注意事项
- 30-70为正常波动区间
- 50为多空分界线
- 背离信号比超买超卖更可靠
- 适合判断超买超卖

## 成交量分析

### 基本概念
- **放量**：成交量显著增加
- **缩量**：成交量明显减少
- **堆量**：成交量持续放大
- **量价关系**：价格与成交量的配合关系

### 量价关系
**量增价涨**：健康上涨，可能持续
**量增价跌**：抛压沉重，可能下跌
**量减价涨**：量价背离，谨慎追高
**量减价跌**：抛压减轻，可能企稳

### 放量标准
- 温和放量：成交量增加50%-100%
- 明显放量：成交量增加100%-200%
- 巨量放量：成交量增加200%以上

## 布林线（BOLL）

### 计算方法
- 中轨 = N日移动平均线
- 上轨 = 中轨 + K × N日标准差
- 下轨 = 中轨 - K × N日标准差

### 买卖信号
**买入信号：**
- 价格触及下轨反弹
- 价格突破中轨
- 喇叭口开口向上

**卖出信号：**
- 价格触及上轨回落
- 价格跌破中轨
- 喇叭口开口向下

### 注意事项
- 价格在上下轨之间波动
- 喇叭口收缩表示震荡，扩张表示趋势
- K值通常取2

## 均线系统（MA）

### 常用均线
- MA5：5日均线，短期趋势
- MA10：10日均线，短期趋势
- MA20：20日均线，月线，波段操作参考
- MA60：60日均线，季线，中期趋势
- MA120：120日均线，半年线
- MA250：250日均线，年线，长期趋势

### 买卖信号
**买入信号：**
- 短期均线上穿长期均线（金叉）
- 价格站上所有均线
- 多头排列（短、中、长期均线从上到下排列）

**卖出信号：**
- 短期均线下穿长期均线（死叉）
- 价格跌破所有均线
- 空头排列（短、中、长期均线从下到上排列）

### 注意事项
- 均线具有滞后性
- 越长周期的均线越稳定
- 需结合其他指标综合判断

## 综合分析原则

1. **多周期共振**：多个指标同时发出相同信号更可靠
2. **趋势优先**：大趋势服从小趋势
3. **量价配合**：价格变化需要成交量配合
4. **背离信号**：顶背离和底背离是重要预警信号
5. **结合基本面**：技术分析需结合基本面分析
